Manage Data

This page allows you to import a plan or a set of plans from an XML file. Download an example. Download the schema.

Please note that this is an atomic operation – it either succeeds or fails. This means that either everything in the uploaded file is processed and added to the database or no action is taken and the database remains unmodified.

Import changes the database. Validate just checks the file. Long files can take a while to import or validate.
